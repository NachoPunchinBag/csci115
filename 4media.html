HAL-9000 is the local name of a 'QNAP TVS-473' Media Server in Ladysmith, British Columbia. It's namesake is the evil supercomputer aboard the Discovery One in the 1968 film '2001: A Space Odyssey.' In the film, HAL tries to murder the main character David, but ends up getting killed by David while singing "Daisy Bell" ( aka Bicycle Built for Two)." Fun for the whole family!

Plex is the media viewing software being used on HAL-9000, in order to make the experience more seamless and user-friendly. Plex doesn't host any of the media used, if you were to open your own Plex account, it would not contain the media from HAL, just some basic free TV and shows. Using our account, and logging into HAL-9000 using Plex, you are given access to terabytes of free media, with no catch whatsoever. We (the admins) host HAL-9000, because we enjoy collecting cool media and sharing it with our friends.

Yep, Sharing Plex information within our family and friend-group is entirely legal in every countrythat is currently accessing it. Think of it like friends sharing DVDs with each-other; as long as noone is directly profiting from it financially it doesn't violate any copyright laws. The serveradmins own all the hardware that runs the server outright, and own the data stored in it. The adminspay an annual subscription service to Plex (roughly $50 USD / $80 CAD), in order to use thesoftware, and everything is provided for free to the server users. Got any digital media that you'dlike to share? Hit us up and we can add it to HAL-9000's library, as long as it meets our qualitystandards for the server.
